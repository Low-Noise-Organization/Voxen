Guía de Instalación

Instala Voxen en tu sistema en menos de 2 minutos.

Requisitos

  • Java 21+ — Voxen requiere Java 21 o superior. Verifica con java --version.
  • Docker — Necesario para el comando voxen dockerize. Instala Docker Desktop o Docker Engine.
  • Cosign (opcional) — Para firmar imágenes Docker. cosign version.
  • Trivy (opcional) — Para escanear vulnerabilidades. trivy --version.
  • GPG (opcional) — Para firmar artefactos. gpg --version.
💡 Nota: Los requisitos opcionales solo son necesarios si usas las funciones avanzadas. Para build y dockerizado básico, solo necesitas Java 21+ y Docker.

Opción 1: Descargar JAR

Descarga el JAR más reciente desde GitHub Releases:

curl -LO https://github.com/lownoise/voxen/releases/latest/download/voxen.jar
chmod +x voxen.jar
alias voxen='java -jar /ruta/completa/voxen.jar'

Añade el alias a tu ~/.bashrc o ~/.zshrc para usarlo en cualquier terminal.

Opción 2: Docker

Usa la imagen oficial de Voxen:

docker pull ghcr.io/lownoise/voxen:latest
alias voxen='docker run --rm -v "$PWD":/workspace ghcr.io/lownoise/voxen:latest'

Esta imagen incluye Java 21, Voxen, y todas las herramientas de build.

Opción 3: Binario Nativo

Compilado con GraalVM native-image para inicio instantáneo:

curl -LO https://github.com/lownoise/voxen/releases/latest/download/voxen-linux-amd64
chmod +x voxen-linux-amd64
sudo mv voxen-linux-amd64 /usr/local/bin/voxen

Descarga la versión para tu arquitectura: amd64 o arm64.

Opción 4: Compilar desde Código Fuente

git clone https://github.com/lownoise/voxen.git
cd voxen
./gradlew build
java -jar cli/build/libs/cli-*.jar --help

# También puedes generar el fat JAR ejecutable:
./gradlew :cli:fatJar
java -jar cli/build/libs/voxen-*.jar --help

Verificar Instalación

Ejecuta el comando info para verificar que todo funciona:

voxen info

Autocompletado en Terminal

Habilita el autocompletado con tabulación para bash o zsh:

# Generar scripts de completado
./gradlew generateCompletions

# Activarlos en tu shell
source build/completions/voxen_completion.bash   # bash
source build/completions/voxen_completion.zsh    # zsh

# O añádelos a tu .bashrc/.zshrc para que sea permanente
echo "source $(pwd)/build/completions/voxen_completion.bash" >> ~/.bashrc
